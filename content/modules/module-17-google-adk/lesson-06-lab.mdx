# Lab: Multi-Agent Research System with ADK + Vertex AI

This lab puts everything from the module together. You will build a three-tier research system, verify it locally, and optionally deploy it to Vertex AI.

<LabBrief title="Multi-Agent Research System with ADK + Vertex AI" time="2–3 hours">
#### Objective
Build a production-ready three-tier research system using ADK 2.0:
- **Root orchestrator** (`LlmAgent`) decomposes the research question and delegates
- **Research pipeline** (`SequentialAgent`): planner → parallel searchers (`ParallelAgent` with 2 branches) → synthesiser
- **Critic** (`LlmAgent`) reviews the synthesis and writes its verdict to session state
- `DatabaseSessionService` with SQLite for persistent state across restarts
- Local verification with `adk dev`, then optional deploy to Vertex AI

#### Setup
```bash
pip install google-adk google-cloud-aiplatform[adk]
export GOOGLE_API_KEY="your-google-ai-studio-key"
# For Vertex AI deployment (optional):
# gcloud auth application-default login
# export GOOGLE_CLOUD_PROJECT="your-project-id"
```

#### Deliverables
- `agent.py` — all agent definitions (orchestrator, pipeline, critic)
- `tools.py` — research tools (web search stub, summariser, fact-checker)
- `session.py` — session service setup with `DatabaseSessionService`
- `runner.py` — async runner loop with event printing
- `adk_config.yaml` — Vertex AI deployment config
- `README.md` — diagram of agent tree + one paragraph on A2A

#### Hard constraints
- Must use `SequentialAgent` for the inner pipeline (not `LlmAgent` for that part)
- Must use `ParallelAgent` for the two search branches
- Must use `DatabaseSessionService(db_url="sqlite:///./sessions.db")`
- All tools must return structured dicts (not plain strings)
- Critic must write its verdict: `tool_context.state["critic_verdict"] = verdict`
</LabBrief>

## Agent tree overview

Before writing code, map the system you are building:

```
root_orchestrator (LlmAgent)
│  Receives the user's research question
│  Delegates to research_pipeline, then to critic
│
├── research_pipeline (SequentialAgent)
│   │  Runs three stages in order
│   │
│   ├── planner (LlmAgent)
│   │     Breaks question into sub-questions
│   │     Writes: session.state["research_plan"]
│   │
│   ├── parallel_searchers (ParallelAgent)
│   │   ├── web_searcher (LlmAgent)
│   │   │     Searches the web for first sub-question
│   │   │     Writes: session.state["web_results"]
│   │   └── academic_searcher (LlmAgent)
│   │         Searches academic sources for second sub-question
│   │         Writes: session.state["academic_results"]
│   │
│   └── synthesiser (LlmAgent)
│         Combines web and academic results into a draft
│         Writes: session.state["synthesis_draft"]
│
└── critic (LlmAgent)
      Reviews the synthesis for gaps and accuracy
      Writes: session.state["critic_verdict"]
```

```python
# session.py — session service setup with DatabaseSessionService

import asyncio
from google.adk.sessions import DatabaseSessionService

# CONSTRAINT: Must use DatabaseSessionService, not InMemorySessionService
session_service = DatabaseSessionService(db_url="sqlite:///./sessions.db")


async def get_or_create_session(user_id: str, app_name: str = "research_app"):
    """Get an existing session for this user or create a new one.
    
    Uses a stable session ID so sessions persist across restarts.
    """
    import hashlib
    session_id = hashlib.sha256(f"{app_name}:{user_id}".encode()).hexdigest()[:32]

    # Try to retrieve existing session
    session = await session_service.get_session(
        app_name=app_name,
        user_id=user_id,
        session_id=session_id,
    )

    if session is None:
        session = await session_service.create_session(
            app_name=app_name,
            user_id=user_id,
            session_id=session_id,
        )
        print(f"Created new session: {session.id}")
    else:
        print(f"Resumed existing session: {session.id}")

    return session
```

### runner.py

```python
# runner.py — async runner loop

import asyncio
from google.adk.runners import Runner
from google.genai.types import Content, Part
from agent import root_orchestrator
from session import session_service, get_or_create_session


async def run_research(question: str, user_id: str = "demo_user"):
    """Run the research pipeline for a given question."""
    session = await get_or_create_session(user_id=user_id)

    runner = Runner(
        agent=root_orchestrator,
        app_name="research_app",
        session_service=session_service,
    )

    message = Content(parts=[Part(text=question)])

    print(f"\nResearching: {question}")
    print("─" * 60)

    async for event in runner.run_async(
        user_id=user_id,
        session_id=session.id,
        new_message=message,
    ):
        # Print intermediate events for visibility
        if hasattr(event, "author") and event.author:
            event_type = type(event).__name__
            print(f"  [{event.author}] {event_type}")

        if event.is_final_response():
            text = ""
            if event.content and event.content.parts:
                text = event.content.parts[0].text
            print(f"\nFinal Answer:\n{text}")

            # Print the critic's verdict from session state
            final_session = await session_service.get_session(
                app_name="research_app",
                user_id=user_id,
                session_id=session.id,
            )
            if final_session:
                verdict = final_session.state.get("critic_verdict")
                if verdict:
                    print(f"\nCritic verdict: {verdict.get('verdict', 'unknown')}")
                    if verdict.get("issues"):
                        print(f"Issues: {verdict['issues']}")


if __name__ == "__main__":
    question = "What is the current state of research into large language model reasoning?"
    asyncio.run(run_research(question))
```
