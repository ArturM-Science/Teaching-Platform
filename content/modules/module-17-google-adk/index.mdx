# Google Agent Development Kit (ADK)

**Framework Elective · Optional · Prerequisite: Module 06**

Google's Agent Development Kit (ADK 2.0) is an engineering-first framework built around hierarchical agent trees and the A2A (Agent-to-Agent) protocol. It integrates natively with Vertex AI for one-command managed deployment and supports cross-framework invocation — an ADK agent can call a LangGraph or CrewAI agent as a peer.
