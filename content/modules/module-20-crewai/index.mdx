# CrewAI

**Framework Elective · Optional · Prerequisite: Module 06**

CrewAI is the lowest-barrier-to-entry framework in this track. You describe who each agent is (role, goal, backstory), what they must do (Tasks), and how the team works (Process). CrewAI handles orchestration — no routing logic, no graph wiring. The hierarchical process automatically creates a manager that delegates, validates, and re-assigns. This module also covers when CrewAI's abstraction level works against you.
