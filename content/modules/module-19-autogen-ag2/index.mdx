# AutoGen / AG2

**Framework Elective · Optional · Prerequisite: Module 06**

AutoGen (now AG2 in its community fork) pioneered the conversational multi-agent model: agents take turns speaking in a shared thread, and a GroupChatManager decides who speaks next. AG2 adds event-driven architecture and explicit DiGraph workflows for production use. Both are covered in this module.
