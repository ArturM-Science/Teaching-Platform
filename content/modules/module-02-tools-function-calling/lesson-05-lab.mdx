# Lab · Multi-Tool Agent with Sandboxed Code Runner

<LabBrief title="Multi-Tool Agent" time="2–3 hours">

#### Objective

Build a working agent with three tools: web search, scoped file access, and a sandboxed code runner. Every tool must handle errors gracefully and have a JSON schema that rejects at least one class of invalid input.

#### Setup

```bash
pip install openai httpx jsonschema
export OPENAI_API_KEY="sk-..."
```

Create a workspace directory the file tool will be scoped to:

```bash
mkdir -p ./workspace
```

#### Deliverables

- `tools.py` — all three tools with schemas and validation
- `agent.py` — the calling loop with MAX_STEPS and error handling
- `test_tools.py` — at least one test per tool showing it rejects an invalid input
- One paragraph in a `README.md` explaining the blast radius of each tool and how you bounded it

#### Hard constraints

- The file tool must reject any path that resolves outside `./workspace/`
- The code runner must time out after 10 seconds maximum
- No tool may raise an unhandled Python exception — all errors must return a structured error dict
- MAX_STEPS must be set and must be no more than 15

</LabBrief>

```python
# agent.py
import json
import openai
from tools import TOOLS, TOOL_MAP

client = openai.OpenAI()
MAX_STEPS = 10

def run_agent(user_message: str) -> str:
    messages = [{"role": "user", "content": user_message}]

    for step in range(MAX_STEPS):
        response = client.chat.completions.create(
            model="gpt-4o",
            messages=messages,
            tools=TOOLS,
        )
        msg = response.choices[0].message
        messages.append(msg)

        if not msg.tool_calls:
            return msg.content

        for call in msg.tool_calls:
            args = json.loads(call.function.arguments)
            # TODO: add argument validation before executing
            result = TOOL_MAP[call.function.name](**args)
            messages.append({
                "role": "tool",
                "tool_call_id": call.id,
                "content": json.dumps(result)
            })

    return "Agent reached maximum steps without completing."

if __name__ == "__main__":
    print(run_agent("Search for the latest Python release and save a summary to workspace/python_news.txt"))
```
