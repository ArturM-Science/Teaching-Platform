# Module 1 · Prompting & Reasoning

**Part 1 — Foundations** · 5 lessons · ~5 hours total

<LearningObjectives>
- Design system prompts with explicit role, task, constraints, and output format sections
- Apply Chain-of-Thought to reduce reasoning errors and understand why it works
- Implement the ReAct pattern — interleaved thinking and action — as a prompt-level design
- Enforce structured JSON output with Pydantic and recover gracefully when validation fails
- Chain prompts across multiple steps with typed state passing between each hand-off
- Build a self-correcting feedback loop that retries on its own failures
</LearningObjectives>

## What you'll build

A reasoning agent that takes a multi-step research problem, works through it using Chain-of-Thought internally, returns a validated Pydantic output, and self-corrects when the output fails validation — passing its own error back as context and retrying. No frameworks. Every step is visible in the message log.
