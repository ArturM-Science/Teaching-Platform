# Module 0 - Mental Models & the Bare-Metal Agent

**Part 1 - Foundations** - 8 lessons - ~5.5 hours total

<LearningObjectives>
- Explain how LLMs generate text: tokens, context windows, temperature, and sampling
- Describe why LLMs hallucinate and under what conditions it gets worse
- Articulate the core agent loop: perceive -> reason -> act -> observe -> repeat
- Choose the right starting point for a first agent: plain Python before frameworks
- Build a working agent in ~50 lines of plain Python with no framework
</LearningObjectives>

## What you'll build

A working agent in ~50 lines of Python that takes a user goal, makes an LLM call, parses a tool call from the response, executes the tool, and loops until done or stuck. No LangChain. No framework. Pure API calls and Python.

This is the foundation everything else builds on. If you understand every line of this agent, every framework you encounter later will make sense because you'll see what it is abstracting.
