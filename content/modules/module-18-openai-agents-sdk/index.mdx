# OpenAI Agents SDK

**Framework Elective · Optional · Prerequisite: Module 06**

The OpenAI Agents SDK (evolved from Swarm) is OpenAI's official framework for production multi-agent systems. Its core primitives are Agents, Handoffs, and Guardrails — plus built-in tracing that integrates with the OpenAI dashboard. It is model-agnostic through LiteLLM and supports structured outputs natively.
