# Course Lab Workspaces

This directory contains lightweight starter workspaces for labs that ask learners to work outside the MDX lesson page.

The lesson pages remain the source of truth for grading criteria and deliverables. These folders provide a practical starting point so commands like `cd labs/module-09-observability-cost-reliability` resolve inside the repository.

Recommended learner flow:

1. Open the lesson page first.
2. Copy or fork the relevant starter workspace.
3. Create the deliverables requested by the lesson.
4. Keep API keys in local environment variables, never in committed files.
